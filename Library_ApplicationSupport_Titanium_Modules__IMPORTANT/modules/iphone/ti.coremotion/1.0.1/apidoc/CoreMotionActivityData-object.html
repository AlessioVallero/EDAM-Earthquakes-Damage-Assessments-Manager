CoreMotionActivityData

Object
Platform Since
iPhone 1.0.0
iPad 1.0.0

Description

Represents the current motion associated with the device.

Note that there is no timestamp property for this class.

Properties

activity : CoreMotionActivity read-only

The motion activity object that defines the current type of motion for the device.

Methods

This type has no methods.

This type has no events.
